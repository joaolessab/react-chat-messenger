### React Chat Messenger for Cognite
 
# Technologies that is being used:
- React.js;
- Node.js;
- Socket.js;

# Running projects:
- Server: `npm start`
- Client: `npm start`

# Acessing projects:
- Server: "http://localhost:5000/";
- Client: "http://localhost:3000/";

# Documentation:
- Socket.io: "https://socket.io/get-started/chat";
